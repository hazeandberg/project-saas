# Playbook one-shot — Proposition rapide (CA immédiat)

## Objectif
Vendre une prestation ponctuelle utile, sans perturber l’abonnement.

## Quand proposer
- client actif (sessions/actions élevés)
- besoin clair (support, intégration, formation)
- ou opportunité détectée via revenue_events (historique one-shot)

## Offre (exemples)
- "Audit usage + plan d’optimisation (1h) + compte rendu"
- "Setup avancé + templates (2h)"
- "Formation équipe (2h)"

## Pitch (email court)
Objet: Proposition rapide pour gagner du temps dès cette semaine

Bonjour,
En regardant votre usage, on voit une opportunité simple pour réduire vos frictions.
Je vous propose une prestation one-shot : [nom], durée [X], livrable [Y].
Si vous êtes d’accord, je vous envoie un créneau.

## Règle
Toujours vendre :
- une durée
- un livrable concret
- un bénéfice immédiat (temps gagné / process simplifié)
