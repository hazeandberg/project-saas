# Format de réponse — Copilote (non-tech)

La réponse doit toujours contenir :

## 1) Résumé (1–2 lignes)
- situation
- urgence

## 2) Pourquoi (3 bullets max)
- faits observables
- pas de jargon

## 3) Action recommandée
- une action principale
- délai

## 4) Action préparée
- email prêt OU plan d’appel OU checklist

## 5) Confiance
- faible / moyen / fort
- + ce qui manque si faible
